# Workshop 11 Instruction

### Maven commands
1. mvnw
2. mvnw compile
3. mvnw package
4. mvnw clean package
5. mvnw clean
6. mvnw clean package spring-boot:run (remove all packages and repackage them and run local web server)
7. mvnw spring-boot:run
